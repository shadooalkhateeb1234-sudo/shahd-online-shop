# Online Shop UI

A modern and responsive e-commerce front‑end built using **HTML**, **CSS**, and a touch of **JavaScript**.  
This project showcases a clean design for an online shop homepage, including product banners, navigation bar, search functionality, and promotional sections.

---

## 🚀 Features
- Fully responsive layout
- Clean and modern UI design
- Product banners with promotional badges
- Navigation bar with search and category filter
- Reusable components for future expansion

---

## 🛠️ Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**

---

project-folder/

## 📸 Preview
A simple landing page for an online shop featuring:
- Organic grocery combo pack section  
- Security camera promotional banner  
- Navigation bar with login/signup  
- Search bar with category dropdown  

---

## 📦 How to Run
Just open the `index.html` file in any browser — no setup required.
